地図で買い物リスト（以下「本アプリ」）における情報の取扱いについて説明します。本アプリは、ユーザーのプライバシーを尊重し、個人情報保護法その他の関連法令を遵守します。

## 1. アプリが取得する情報

- 端末の位置情報（緯度・経度）: 登録地点への出入りの検知、および停止時に近隣の候補店舗を評価するために利用します。近隣候補通知や地点検索のため、Google Places API などの外部サービスへ送信されることがあります。
- 身体活動情報（Activity Recognition）: ユーザーが一定時間停止したことを検知し、近隣候補通知のタイミング制御に利用します。身体活動情報そのものを外部サービスへ送信することはありません。
- アプリ内で登録した買い物アイテム、地点の名称・座標・メモ: 端末内データベースに保存します。このうち未紐付けアイテムの名称は、近隣候補通知のカテゴリ判定のために外部サービスへ送信されることがあります。
- 近接通知の診断ログ: 登録処理、イベント受信、カテゴリAPI呼び出し、Places検索の結果などを端末内に保持することがあります。ログはアプリ内から確認・共有・削除でき、通常利用時に自動で外部送信は行いません。

## 2. 位置情報の取り扱い

- 利用目的: 登録済み地点（半径100m）への出入りを検知し、また未紐付けアイテムについて近隣で買えそうなお店を推定して通知するために使用します。
- 取得タイミング: アプリがバックグラウンド状態または終了している場合でも、OS の Geofencing 機能および Activity Recognition により検知が行われます。
- 外部サービス利用: 近隣候補通知や地点検索では、Google Places API または当アプリ開発者が管理する API に対して、位置情報、検索語、アイテム名、言語設定、国コードなど、必要最小限の情報を送信することがあります。
- 保存期間: 位置情報の履歴は保存せず、通知制御に必要な最小限の情報のみ保持します。重複通知を避けるため、通知時刻と通知時点の座標、関連する候補店舗IDを端末内に保持することがあります。
- 権限設定: 位置情報（常に許可またはアプリ使用中のみ）、身体活動、通知権限を必要に応じてリクエストします。端末設定からいつでも変更・削除できます。

## 3. データの第三者提供

本アプリは、広告目的でユーザーデータを第三者へ販売・提供することはありません。一方で、機能提供のために以下の外部サービスを利用します。

- Google Places API: 地点検索および近隣候補店舗の検索のため
- 当アプリ開発者が管理する API: 買い物アイテム名から候補店舗カテゴリを推定するため

これらのサービスには、機能提供に必要な範囲で位置情報、検索語、アイテム名、言語設定、国コードなどが送信されることがあります。ただし、身体活動情報そのものを外部サービスへ送信することはありません。

また、以下の場合に限り、法令に基づき情報を開示することがあります。

- 法令に基づく要請がある場合
- 犯罪捜査・裁判手続など、公的機関からの正当な照会があった場合
- 人の生命・身体・財産の保護のために必要であり、ユーザーの同意を得ることが困難な場合

## 4. データの管理

登録した買い物アイテム、地点情報、通知制御用の状態、近接通知の診断ログは主に端末内に保存され、アンインストール時に自動的に削除されます。クラウド同期やバックアップ機能は提供していません。外部サービスへ送信されたデータは、それぞれのサービス提供者または当アプリ開発者管理 API の運用方針に従って処理されます。端末のセキュリティ設定を適切に管理することをおすすめします。

## 5. ユーザーの権利

登録したアイテムや地点情報はいつでも編集・削除できます。位置情報・身体活動・通知権限を拒否した場合、一部機能（地点付近での通知、近隣候補通知など）が利用できなくなることがあります。

## 7. ポリシーの変更

本ポリシーの内容を変更する場合は、アプリ内のお知らせまたはリリースノートで告知します。変更後もアプリを継続して利用する場合、改定後の内容に同意したものとみなします。
