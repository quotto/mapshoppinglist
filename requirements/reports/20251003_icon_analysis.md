# アイコンセット分析レポート（2025-10-03）

## 対象
- `app/src/main/res/mipmap-*/ic_launcher_background.webp`
- `app/src/main/res/mipmap-*/ic_launcher_foreground.webp`
- 出力用に `sips` でBMPへ変換し、Pythonスクリプトで画素解析。

## 色彩分析
### 背景グラデーション
- 主成分: 深い青の縦方向グラデーション。
  - 上部平均色: `#1B7CB0`
  - 中央平均色: `#1B70A0`
  - 下部平均色: `#1B6491`
- 近似パレット: `#1B7CB0`, `#1B70A0`, `#1B6491`, `#1B5B86`
  - 明度の低い青ほど下側に配置されており、空から地図を見下ろすような奥行きを演出。

### 前景シンボル
- 優勢色: 赤系 `#EA5F52` 周辺（輝度差で最大約3%のバリエーション）。
- 補助色: ライトグレー〜ホワイト `#F6F4F3` (`#F5F4F2`)。
- 色構成比（おおよそ）
  - レッド系: 約60%
  - ハイライト（白系）: 約25%
  - その他（影・アンチエイリアス）: 約15%

## 形状・モチーフ
- 中央シンボルは地図ピンを想起させるティアドロップ形状。
  - 白系ハイライトが中央部に配置され、リスト用紙やチェックリストを連想させる幾何学パターンを構成。
  - ピン外周部の赤は軽いドロップシャドウ処理が施され、背景の青と十分なコントラストを形成。
- 全体は角丸のボーダーで囲われたフレーム感があり、グリッド調の余白を保った現代的フラットデザイン。

## ブランドカラー抽出
- **ブランド基調色（ディープブルー）**: `#1B70A0` を中心に、トップライト `#1B7CB0`, ボトムシェード `#1B6491` を組み合わせる。
- **アクセントカラー（ピンレッド）**: `#EA5F52`
- **サブアクセント（ハイライト）**: `#F6F4F3`
- **コントラスト補助**: テキスト用に `#0F1C2B`（ブルーのシャドウトーンから抽出）を検討。

## UIデザインへの示唆
1. **配色**
   - アプリ全体のベースカラー: `#1B70A0` をキーカラーに設定し、ダーク/ライト状態で明度を調整。
   - CTAや強調表示にはアクセントレッド `#EA5F52` を使用。ただし連続使用を避け、行動誘発要素に限定。
   - 背景グラデーションは画面ヒーロー領域やスプラッシュで活用し、通常画面ではフラットな `#F6FAFD` など明度の高いバリアントを検討。

2. **コンポーネント形状**
   - ピンのティアドロップ形状と角丸矩形をUIのラウンド率ガイドとして参照（例: 16dp～24dp 角丸）。
   - ボタンやカードに僅かなシャドウを付け、前景アイコンの立体感と統一する。

3. **アイコンスタイル**
   - マテリアルアイコンをそのまま使うのではなく、単色塗り＋円形/角丸背景でブランドカラーを適用。
   - 状態アイコン（通知、権限要求など）にはレッドアクセントを局所的に使用し、注意喚起効果を高める。

4. **タイポグラフィ**
   - 本文フォント: DM Sans（既定）。階層構造を作るため、見出しは Medium/Bold、本文は Regular を基本とし、`lineHeight` を1.3〜1.4程度に設定。
   - 背景ブルー上の白文字は WCAG AA を満たすようウェイトとサイズを調整。
